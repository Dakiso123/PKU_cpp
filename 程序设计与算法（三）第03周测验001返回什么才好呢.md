填空类的题目我们先看主函数：

```cpp
	int m,n;
	A a;		//定义了类A的对象a
	cout << a.val << endl;		
//输出a.val的值
//在这里我们注意到，定义的a并没有给他值
//所以这里生效的是构造函数初始化
	while(cin >> m >> n) {//输入m n
		a.GetObj() = m;	
        //这个函数类里面没有，需要我们写
        //这种样式的赋值，我们可以联想到引用
		cout << a.val << endl;//输出的是m
		a.GetObj() = A(n);//这个函数的返回构造函数Ａ的结果
		cout << a.val<< endl;
	}
	return 0;
```

所以，我们可以这样写：

```cpp
class A {
public:
	int val;
	A(int n=123)
    {
        val = n;
    }
    A & GetObj()
    {
        return * this;
}

};
```
